1. Создал файл AGENTS.md в котором описал доменные сущности
    ```

    - Публичный сценарий гостя

    Доменная сущность "владелец" содержит следующие поля
    - id
    - Имя

    Доменная сущность "тип события" содержит следующие поля
    - id
    - название
    - длительность события в минутах

    Доменная сущность "слот" содержит следующие поля
    - id
    - ссылка на тип события
    - ссылка на владельца
    - ссылка на публичный сценарий гостя
    - дата и время в часах и минутах

    Доменная сущность "бронирование" содержит следующие поля
    - id
    - дата и время в часах и минутах

    - ссылка на владельца
    - ссылка на публичный сценарий гостя

    Доменная сущность "публичный сценарий гостя" содержит следующие поля
    - id
    - имя
